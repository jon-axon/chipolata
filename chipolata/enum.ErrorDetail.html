    UnknownInstruction {

    },

    },

        key: u8,

    },

        new_state: ProcessorStatus,
    },
    UnknownError,
}
Expand description

An Error enum used throughout the Chipolata crate to communicate details of runtime errors that have occurred.

Instances of ErrorDetail are bubbled-up to the hosting application through the public API methods.
